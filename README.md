# Talento Sin Fronteras - Backend

## Estado actual
Estructura base del backend creada con arquitectura MVC.

## Estructura del proyecto
```
backend/
├── src/
│   ├── controllers/
│   ├── models/
│   ├── routes/
│   └── middlewares/
```

## Tecnologías
- Node.js + Express
- MongoDB Atlas + Mongoose
- JWT + bcryptjs

## En progreso
- Conexión a MongoDB Atlas
- Autenticación (registro y login con JWT)

## Pendiente
